Tienda online Java - SQL


    • Es un programa hecho con java que almacena:
    - Clientes (premium y estándar, cada uno con sus respectivos descuentos automáticos).
    - Artículos (cada uno con sus gastos de envió y tiempo de producción).
    - Pedidos (registran artículos y un cliente), calculan el tiempo de producción total de los mismos y registras si están enviados(no se pueden enviar) o no enviados.
    El programa almacena los datos en una base de datos SQL y es capaz de generar tickets detallados. Se han utilizado librerías como JUnit (para hacer tests) y el ORM hibernate.
